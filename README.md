# WisperTalk Releases

## Demo



https://github.com/user-attachments/assets/bab39c98-a6c3-4a98-a551-05f2263a0449




Public release artifacts for [WisperTalk](https://wispertalk.com), the voice-dictation app for Windows and macOS.

This repo holds **only built binaries** — `.exe` for Windows, `.dmg` and `.zip` for macOS, plus autoupdater feeds.

The application source lives in the private [`wispertalk-app`](https://github.com/bensblueprints/wispertalk-app) repo. CI on each release publishes the built artifacts here.
